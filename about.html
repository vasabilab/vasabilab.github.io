About vasabiLab

The objectives of vasabiLab are the followings:

  • Develop innivative software systems for cloud computing
  • Distribute knowledge about cloud computing, software systems, and distributed systems
  • Develop tools to promote the adoption of open source cloud computing systems such as OpenStack and qemu-kvm
  • Provide tutorials and education materials to teach open source cloud software
